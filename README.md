# UDP Connectivity Test Tool

A small Python utility for controlled UDP packet testing in authorized lab environments.

## Purpose

This tool is intended only for:
- local network diagnostics
- learning basic UDP socket behavior
- testing your own systems in a private lab
- verifying firewall or port reachability with permission

## Important Notice

Use this tool only on systems and networks you own or have explicit written permission to test.

Do not use it to:
- disrupt services
- send traffic to third-party systems without consent
- generate excessive traffic
- bypass security controls
- perform denial-of-service behavior

## Requirements

- Python 3.x

## How to Run

1. Open a terminal.
2. Run the script with Python 3.
3. Enter the destination IP and port only for a system you are authorized to test.

## Safety Rules

- Keep tests limited and controlled.
- Use only on localhost, a private VM, or a closed lab network.
- Stop immediately if you notice unexpected network impact.
- Do not run continuous high-rate sending.
- Respect local laws, network policies, and acceptable-use rules.

## Educational Use

This program can help you understand:
- UDP sockets
- packet sending basics
- simple command-line input handling
- error handling in Python
